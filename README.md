# Voxel Landscape

Procedural voxel landscape engine

![Flyover gif](https://raw.githubusercontent.com/qwikdraw/voxel-landscape/master/screenshots/voxel.gif)

### Requirements
`pkg-config`  
`glm`  
`glfw`  
`wget`  
`make`  

### Building

On macOS:
```
./deps.sh
make
```
